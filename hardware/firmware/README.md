# JalRakshak Firmware

This folder contains the ESP32 firmware for the JalRakshak water monitoring system.

## Planned Features of JalRakshak

- Read water temperature using DS18B20
- Measure water level using HC-SR04 ultrasonic sensor
- Monitor water quality using TDS sensor
- Detect turbidity levels
- Monitor pH levels
- Send sensor data to the JalRakshak backend
- Trigger alerts when unsafe conditions are detected
