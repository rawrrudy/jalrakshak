# JalRakshak Schematic

This folder contains the KiCad schematic files for JalRakshak.

## Sensors

- DS18B20 Temperature Sensor
- HC-SR04 Water Level Sensor
- TDS Sensor
- Turbidity Sensor
- pH Sensor

## Controller

- ESP32 DevKit V1

The schematic defines all electrical connections between the ESP32 and sensors.
