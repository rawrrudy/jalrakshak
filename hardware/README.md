# JalRakshak Hardware

## Hardware Components

- ESP32-WROOM-32
- pH Sensor
- DS18B20 Temperature Sensor
- HC-SR04 Ultrasonic Sensor (Water Level Monitoring)
- TDS Sensor
- Turbidity Sensor

## Parameters Monitored

- pH
- Total Dissolved Solids (TDS)
- Turbidity
- Water Level
- Water Temperature

## PCB Design

The PCB was designed using KiCad and integrates all sensors with the ESP32 for real-time data acquisition and transmission.

## Files

- 'JalRakshak V1.kicad_sch' - Schematic
- 'JalRakshak V1.kicad_pcb' - PCB Layout
- 'JalRakshak V1.kicad_pro` - KiCad Project File
- Gerber files for PCB manufacturing
